# 2026-07-07 — Personal Website Redesign (shipped)

Full rebuild of the personal site, merged to `master` and live at https://loiht2.vercel.app.

## What was built

- **Stack:** Next.js 16 (App Router) + React 19, Tailwind CSS v4, Geist fonts, `next-themes` (dark-first, AA-contrast palette). Deployed Vercel-native (dropped static export).
- **Animation:** CSS-first — entrance choreography, scroll-driven reveals (`animation-timeline`), view-transition page fades. No animation library; content stays visible without JS.
- **Pages:** Home (hero, news, featured projects, publications, static photo grid), Research (new), Projects, Writing (MDX pipeline via `next-mdx-remote` + `gray-matter`), print-safe data-driven CV, custom 404.
- **Content:** all data in `content/*.js` modules; unfilled fields marked `FILL_ME` and hidden until set (no placeholder text).
- **SEO:** generated sitemap / robots / manifest / RSS / OG image / icons, all on the Vercel URL, per-page canonicals. Security headers + legacy redirects (`/blog`, `/posts/*`) in `next.config.mjs`.
